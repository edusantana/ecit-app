# ecit-app

Sistema auxiliar Escola Cidadã Integral Técnica (ecit)

# Funcionalidades

## Escolha da Eletiva

Os alunos podem escolher uma eletiva, entre as que ainda tem vaga, qual delas deseja cursar no semestre.

### Caso de sucesso

- Sistema lista eletivas com suas respectivas vagas (aleatoriamente)
- Aluno escolhe uma disciplina
- Registra sua inscrição nela (selecionando seu nome e uma senha)

## Escolha do tutor

Escolhem dois professores, com primeira e segunda opção, joga para os da segunda opção.
